# 그룹팅 웹사이트

그룹팅 모바일 앱을 위한 공식 랜딩 페이지입니다.

## 📁 파일 구조

```
website/
├── index.html      # 메인 HTML 페이지
├── style.css       # 스타일시트
├── script.js       # 인터랙티브 기능
└── README.md       # 이 문서
```

## 🎨 디자인 특징

### 현대적인 디자인 시스템
- **그라디언트 색상**: 보라색(#667eea)에서 분홍색(#f5576c)까지 생동감 있는 색상 팔레트

- **부드러운 애니메이션**: 스크롤 애니메이션, hover 효과, 페이드인 효과
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 완벽 지원

## 🚀 실행 방법

### 1. 로컬에서 바로 열기
```bash
# 파일 탐색기에서 index.html 더블클릭
# 또는 브라우저에서 직접 열기
```

### 2. 로컬 서버로 실행 (권장)
```bash

cd website
python -m http.server 8000

# 브라우저에서 http://localhost:8000 접속
```

### 3. VS Code Live Server 사용
```bash
# VS Code에서 index.html 우클릭 -> "Open with Live Server"
```

## ✨ 인터랙티브 기능

### 애니메이션
- **스크롤 애니메이션**: 섹션이 화면에 나타날 때 페이드인
- **통계 카운터**: Hero 섹션의 숫자가 자동으로 카운트업
- **패럴랙스 효과**: 배경 오브가 스크롤에 따라 움직임
- **3D 카드 틸트**: 마우스 움직임에 따라 카드가 기울어짐

### 사용자 경험
- **부드러운 스크롤**: 네비게이션 링크 클릭 시 smooth scroll
- **네비게이션 바**: 스크롤 시 숨겨지고 나타남
- **알림 시스템**: 버튼 클릭 시 우측 상단에 알림 표시
- **이스터에그**: 코나미 코드 입력 시 특별 효과!

## 🎮 숨겨진 기능

### 코나미 코드
키보드로 다음 순서대로 입력하세요:

## 🎯 커스터마이징

### 색상 변경
`style.css`의 CSS 변수를 수정하세요:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #f5576c;
    /* 원하는 색상으로 변경 */
}
```

### 텍스트 수정
`index.html`에서 직접 텍스트를 수정할 수 있습니다.

### 이미지 추가
- `../assets/` 폴더에 이미지를 추가하고
- HTML에서 경로를 참조하세요
